# Installation

Always clone the latest release, not the latest commit. Commits can be `[UNSAFE]`, but releases are not.

Clone the repo:

```bash
git clone https://github.com/Mkorp-Official/Krypt.git
```

Change your directory:

```bash
cd path/to/krypt
```

Install the script:

```bash
npm install -global
```

Start the process:

```bash
krypt
```
